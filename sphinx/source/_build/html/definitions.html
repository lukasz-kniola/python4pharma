Dataframe¶

Many (but not all) operations on dataframes will return a new dataframe transformed according to the function used, but not affect the original dataframe. To apply the method to the original dataframe, use inplace=True option.

For example:

df.sort_values("NAME")

will return a new dataframe (in this case not assigned to any var). To make the sort permanent we need to assign the returning daraframe back to the same name, or use the inplace option:

df = df.sort_values("NAME")

or

df.sort_values("NAME", inplace=True)
