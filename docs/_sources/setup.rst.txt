Setup
******************************

:strong:`Installing Pandas`

You can install Pandas using pip or conda. Whichever is your favourite.

.. code-block:: python

    pip install pandas

or 

.. code-block:: python

    conda install pandas

:strong:`Importing module`

.. code-block:: python

    import pandas as pd

    # it may also be useful to import the numpy module
    import numpy as np
